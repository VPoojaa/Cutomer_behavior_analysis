# 🛍️ Customer Shopping Behavior Analysis

## 📌 Project Overview
This project analyzes customer shopping behavior using a dataset of 3,900 transactions across multiple product categories.  
The goal is to understand purchasing patterns, customer segments, and generate meaningful business insights.

---

## 🎯 Objective
- Analyze customer purchase behavior  
- Identify top-performing products and categories  
- Understand customer segmentation  
- Support data-driven decision making  

---

## 📊 Dataset Overview
- Total Transactions: 3,900  
- Features: 18  
- Includes customer demographics, product details, and purchase behavior  

---

## 🛠 Tools & Technologies Used
- Python  
- SQL  
- Power BI  

---

## 📊 Key Analysis Performed
- Revenue analysis by gender  
- Product category performance  
- Customer segmentation (Loyal, Returning, New)  
- Sales trends and purchasing patterns  

---

## 📈 Key Insights
- Male customers contribute majority of the revenue  
- Clothing category generates the highest sales  
- Most customers are repeat (loyal) buyers  
- Different age groups show varied purchasing behavior  

---

## 🚀 Conclusion
This project provides insights into customer behavior and helps businesses improve marketing strategies, customer engagement, and overall sales performance.
